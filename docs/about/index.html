# About

Caliban is a project developed by Pierre Ricadat aka @ghostdogpr.

The name is inspired by the SF novel and tv series The Expanse.

Thanks:

  • @deckerst for the logo
  • @yoohaemin, @adamgfraser and @jdegoes for the early feedback
  • the Morpheus library in Haskell for the inspiration leading to the GraphQL backend implementation
  • the elm-graphql library for the inspiration leading to the GraphQL client implementation
  • all Caliban's contributors!
